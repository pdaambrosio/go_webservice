# go_webservice

Go Webservice

## Description

This is a simple webservice written in Go.

## Requirements

- Go 1.20

## Installation

```bash
go get github.com/pdaambrosio/go_webservice
```

## Usage

```bash
go run main.go
```

or

```bash
go build main.go
./main
```
